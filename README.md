This is a simple career interest quiz website created using HTML, CSS, and JavaScript.  
I built this project to help 10th class students understand their interests and choose the right stream after school.

The quiz contains 10 easy questions based on student interests.  
After answering all questions, the website suggests a suitable stream like:

- Science
- Commerce
- Arts
- Or Mixed Interests


## üíª Technologies Used

- HTML for structure  
- CSS for design  
- JavaScript for quiz logic and result calculation  



##  Purpose of This Project

As a student learning frontend development, I wanted to create something useful and practical.  
This project is designed to guide students in making better career decisions based on their interests.



##  How It Works

1. Students answer 10 simple interest-based questions.
2. Each question is linked to a stream category.
3. The system calculates the score.
4. The stream with the highest score is suggested.
